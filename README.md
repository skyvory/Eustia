# Eustia

## Description

Eustia-themed watch face featuring:
- Aura border for battery status
- Dynamic background switching between morning, day, afternoon, and night.

Aura indicator (battery percentage):
- 86-100: green
- 41-85: blue
- 21-40: yellow
- 11-20: orange
- 0-10: red
